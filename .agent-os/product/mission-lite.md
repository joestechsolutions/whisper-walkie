# Product Mission (Lite)

Whisper Walkie is a cross-platform push-to-talk speech-to-text app that types directly into any window — 100% local, powered by OpenAI's Whisper, no cloud APIs or subscriptions.

It serves developers, writers, accessibility users, and anyone who wants fast, private voice-to-text on Windows, macOS, or Linux. Unlike cloud-based STT services or clipboard-based local tools, Whisper Walkie injects text via platform-native keystroke APIs so it works everywhere a keyboard works, with zero audio leaving the machine.
