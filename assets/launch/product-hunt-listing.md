## Description
Whisper Walkie is a free, open-source push-to-talk voice typing app that works in any window on your computer.

Hold a key. Speak. Release. Your words appear wherever your cursor is — browser, Slack, VS Code, Word, Discord, anywhere.

**What makes it different:**
- 100% local — your audio never leaves your machine. No cloud APIs, no accounts, no data collection.
- Works in ANY app — not just a browser extension. Types via native keyboard injection so it works everywhere a keyboard works.
- Push-to-talk — only records when you hold the key. No always-listening.
- Free and open source (MIT) — no subscriptions, no freemium limits.

**How it works:**
1. Hold the push-to-talk key (default: Right Alt)
2. Speak into your microphone
3. Release the key
4. Text is typed directly into whatever app has focus

Built with faster-whisper (OpenAI's Whisper model running locally via CTranslate2), Flet for the GUI, and platform-native keyboard APIs for text injection.

GPU acceleration supported on Windows/Linux with NVIDIA CUDA. CPU mode works great on all platforms including macOS.

The Whisper speech model is bundled — download once, works offline forever. No internet connection needed.

**Available for:**
- Windows (installer + portable zip)
- macOS (zip)
- Linux (tar.gz)

Built by Joe's Tech Solutions — we run private AI for small businesses. Whisper Walkie is our gift to the community.
